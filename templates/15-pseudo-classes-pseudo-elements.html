CSS Pseudo-Classes & Pseudo-Elements

1. :hover

.element:hover { ... }

2. :active

.element:active { ... }

3. :focus

.element:focus { ... }

4. :link & :visited

Link belum dikunjungi (:link)

Link yang sudah dikunjungi (:visited)
a:link { ... } a:visited { ... }

5. :checked

input:checked { ... }

6. :disabled

button:disabled { ... } button:enabled { ... }

7. :required & :optional

input:required { ... } input:optional { ... }

8. :valid & :invalid

input:valid { ... } input:invalid { ... }

9. :first-child & :last-child

  • Item pertama (first-child - merah)
  • Item kedua
  • Item ketiga
  • Item terakhir (last-child - hijau)
li:first-child { ... } li:last-child { ... }

10. :nth-child

  • Item 1 (odd)
  • Item 2 (even)
  • Item 3 (nth-child(3) - merah)
  • Item 4 (even)
  • Item 5 (odd)
li:nth-child(odd) { ... } li:nth-child(even) { ... } li:nth-child(3) { ... }

11. :nth-of-type, :first-of-type, :last-of-type

Heading

Paragraf pertama

Paragraf kedua (nth-of-type(2) - biru)

Paragraf ketiga

p:nth-of-type(2) { ... } p:first-of-type { ... } p:last-of-type { ... }

12. :not()

  • Item pertama (bukan :not)
  • Item kedua
  • Item ketiga
li:not(:first-child) { ... }

13. ::before

Teks dengan ::before pseudo-element

.element::before { content: "..."; }

14. ::after

Teks dengan ::after pseudo-element

.element::after { content: "..."; }

15. ::first-letter

Ini adalah contoh ::first-letter pseudo-element. Huruf pertama akan lebih besar dan berwarna biru.

.element::first-letter { ... }

16. ::first-line

Ini adalah baris pertama yang akan di-styled. Baris kedua dan seterusnya akan tetap normal. Resize browser untuk melihat efek pada baris pertama.

.element::first-line { ... }

17. ::selection

Coba select (blok) teks ini untuk melihat efek ::selection pseudo-element.

.element::selection { ... }

18. ::placeholder

.element::placeholder { ... }

19. ::marker

  • Item dengan marker styled
  • Item kedua
  • Item ketiga
li::marker { ... }

20. Combined Pseudo-Classes

Hover, Active, dan Combined States
a:hover { ... } a:active { ... }

21. Input States (Focus, Valid, Invalid)

input:focus { ... } input:valid { ... } input:invalid { ... }
