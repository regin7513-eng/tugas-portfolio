Pengenalan Semantic HTML

Semantic HTML memberikan makna pada elemen HTML, membantu browser dan developer memahami struktur konten.

Apa itu Semantic HTML?

Semantic HTML adalah penggunaan tag HTML yang memiliki makna semantik, seperti <header>, <nav>, <main>, <article>, dan <footer>.
