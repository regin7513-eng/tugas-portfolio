1. Static Position (default)

Static Box
Static Box 2

Static adalah posisi default elemen. Elemen mengalir sesuai urutan dokumen.

2. Relative Position

Normal Box
Relative (left: 30px, top: 20px)
Normal Box 3

Relative memindahkan elemen relatif terhadap posisi normalnya.

3. Absolute Position

Absolute (top: 20px, right: 20px)
Absolute (bottom: 20px, left: 20px)

Absolute memposisikan elemen relatif terhadap elemen parent terdekat yang berposition relative.

4. Fixed Position

Lihat tombol "Back to Top" di pojok kanan bawah. Tombol ini menggunakan position: fixed.

Fixed memposisikan elemen relatif terhadap viewport browser dan tetap di posisi saat scroll.

Sticky Box (scroll ke bawah untuk melihat efek)

Scroll ke bawah untuk melihat efek sticky. Elemen akan menempel di atas saat mencapai posisi tertentu.

6. Z-Index

Z-Index: 1
Z-Index: 2
Z-Index: 3

Z-Index menentukan urutan tumpukan elemen. Semakin tinggi nilai, semakin di atas elemen tersebut.

7. Overflow

overflow: visible (default)

Konten yang melebihi container akan terlihat di luar.

overflow: hidden

Konten yang melebihi container akan disembunyikan.

overflow: scroll

Konten yang melebihi container akan menampilkan scrollbar.

overflow: auto

Scrollbar hanya muncul jika diperlukan.
