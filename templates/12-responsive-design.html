Responsive Design Examples

1. Viewport Meta Tag

Meta tag viewport di head:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Meta tag ini membuat website responsif di berbagai ukuran layar.

2. Fluid Typography

Teks ini menggunakan clamp() untuk ukuran font responsif

Resize browser untuk melihat perubahan ukuran font.

font-size: clamp(1rem, 2.5vw, 2rem);

3. Fluid Width

Container ini menggunakan width: 100% dengan max-width: 800px

Resize browser untuk melihat responsivitasnya.
